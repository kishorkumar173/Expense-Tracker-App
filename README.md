# 💰 Expense Tracker App (Data Science Project)

## 📌 Overview
This project is a data-driven Expense Tracker application built using Python and Streamlit. It allows users to analyze and visualize their spending patterns.

## 🎯 Objective
To help users track expenses, analyze spending behavior, and make better financial decisions.

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit

## 📊 Features
- Category-wise expense analysis
- Monthly spending trends
- Interactive dashboard
- Data filtering options
- Insights generation
